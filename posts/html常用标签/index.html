HTML常用标签

August 30, 2019

a 标签的用法

1. href

包含超链接指向的 URL 或 URL 片段。

2. target

<a href="https://www.baidu.com" target="_blank">first</a>

此时可在新窗口打开百度

其他的还有_top, _parent, _self

3. download

理论上在标签里加上download可以下载，但是很多浏览器不支持

4. rel=noopener

暂空

img 标签的用法

<img src="" alt="" />
  1. 作用

发出 get 请求，展示一张图片

  1. 属性

src 为图片地址， alt 为图片加载失败时，显示的文字

  1. 事件

onload/onerror

  1. 响应式

    max-width: 100%;

table 标签的用法

table {
  width: 300px;
  table-layout: auto;
  border-spacing: 20px;
  border-collapse: collapse;
}

table th,
td {
  border: 1px solid saddlebrown;
}

<table>
  <thead>
    <tr>
      <!-- table row -->
      <th>学科</th>
      <th>男生</th>
      <th>女生</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td>语文</td>
      <td>10人</td>
      <td>15人</td>
    </tr>
    <tr>
      <td>数学</td>
      <td>15人</td>

  </tbody>
  <tfoot>
    <tr>
      <td>总数</td>
      <td>25</td>
      <td>25</td>
    </tr>
  </tfoot>
</table>
